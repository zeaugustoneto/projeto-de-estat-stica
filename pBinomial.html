DISTRIBUIÇÃO DE PROBABILIDADES BINOMIAL

Modelos Teóricos Discretos de Probabilidades.

Distribuição de Probabilidade UNIFORME

Modelo Teórico Contínuo de Probabilidade

Característica:

  • Este modelo admite dois resultados: S = sucesso; e F = fracasso;
  • O sucesso denominamos de p;
  • O fracasso denominados de q;

Descrição:

  • x: 0, 1, 2, 3, 4, ..., n
  • P (x = k) = (n k) * pk * qn-k onde k = 0, 1, ..., n (onde k é o evento e n é o tamanho da amostra)

Observações:

  • Análise Combinatória de n e k (n k) = n! / (n-k)! * k! (! = fatorial)
  • A análise combinatória de (n 0) é igual a 1 | de (n n) é igual a 1 | de (n 1) é igual a n
  • Média - μ(x) = n.p
  • Desvio Padrão - ϑ2(x) = n.p.q , portanto ϑ(x) = √n.p.q

Quando e como uso a distribuição binomial?:

  • Quanto tenho somente duas ocorrências para encontrar a probabilidade de um evento ocorrer, por exemplo, no controle de qualidade de uma empresa, ou as peças são boas ou são com defeito;
  • Se quero saber a probabilidade de ocorrência de peças com defeito o meu sucesso é "com defeito", mas se for de peças sem defeito, o meu sucesso passa a ser as peças "sem defeito". Portanto o meu sucesso é o que quero encontrar e o meu fracasso o resto.
